# Setup tailwindcss with nextjs superfast

```
npx create-next-app my-tailwind-setup
cd my-tailwind-setup

npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
npx tailwindcss init -p #Creates a 'tailwind.config.css' file.
```

Src: https://github.com/vercel/next.js/tree/canary/examples/with-tailwindcss
