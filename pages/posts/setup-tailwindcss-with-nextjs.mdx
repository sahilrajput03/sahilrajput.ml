# Setup tailwindcss with nextjs superfast

- Official Docs: <a target="_blank" href='https://tailwindcss.com/docs/guides/nextjs'>Install Tailwind CSS with Next.js</a>
- NextJS Expample App:  <a target="_blank" href="https://github.com/vercel/next.js/tree/canary/examples/with-tailwindcss">with-tailwincss</a> a nextjs example in official nextjs repo.

ğŸ”½ï¸ Step 2 B: Open `styles/global.css` file, remove import of `globals.css` and add below one::

  color: deeppink;
}
`} />

`}/>
